Project Overview

Perfect combination of innovative technology and humanistic care

Project Background and Significance

Social Context: With the intensification of global population aging, the number of Alzheimer's Disease (AD) patients continues to grow. According to statistics, there are approximately 50 million patients worldwide, expected to reach 150 million by 2050. These patients face problems such as memory loss and emotional apathy, seriously affecting their quality of life.

Technological Innovation: This project pioneered the combination of EEG signal analysis, deep learning, and natural language processing technologies to create a complete emotion recognition and poetry generation system. By monitoring patients' emotional states in real-time and generating personalized poetry content, it helps awaken memories and stabilize emotions.

Application Value: The system can be widely applied in nursing homes, rehabilitation centers, and home care scenarios, providing non-pharmacological emotional support solutions for patients, reducing the burden on caregivers, and improving the quality of life for both patients and their families.

Core Technology Analysis

1. EEG Signal Processing:

  • Multi-channel EEG device for data acquisition
  • Signal preprocessing: filtering, noise reduction, normalization
  • Feature engineering: time domain, frequency domain, time-frequency domain feature extraction

2. Deep Learning Models:

  • CNN Layers: Automatically extract spatial features from EEG signals
  • LSTM Layers: Capture long-term dependencies in time series
  • Dropout: Prevent overfitting and improve generalization
  • Model Accuracy: Training set 95%+, test set 90%+

3. NLP Poetry Generation:

  • Based on Deepseek AI large language model
  • Carefully designed prompt engineering
  • Integrating patient information, emotional states, and memory fragments
  • Generating concise, warm, and personalized modern poetry

Model Training Results

High-performance models trained and optimized with extensive data

CNN-LSTM Hybrid Model

Combining the advantages of convolutional neural networks and long short-term memory networks, demonstrating excellent convergence and stability during training. Validation accuracy reaches 92%, with loss function rapidly decreasing and stable.

  • Training Accuracy: 95.3%
  • Validation Accuracy: 92.1%
  • F1-Score: 0.91

LSTM Sequential Model

Best Model

Long short-term memory networks effectively capture temporal features and long-term dependencies of EEG signals, with excellent modeling capabilities for temporal dynamics of emotional changes. Stable training process with outstanding generalization performance.

  • Training Accuracy: 93.8%
  • Validation Accuracy: 90.5%
  • Recall: 0.89
